2.数组
	在一个变量中存储多个数据

	1.声明数组
		1. var 数组名=['张三','李四','王五'];
		2. var 数组名=new Array('张三','李四','王五');
	
	2.数组使用
		1.获取数组元素
			数组名[下标]
		
		2.获取数组长度
			数组名.length
		
		3.调用不存在的元素，返回undefined
		4.如果给不存在的元素赋值，则会添加元素

	3.遍历数组
		利用循环将数组元素挨个获取一遍
		for(var i=0;i<数组.length;i++){
			//i -- 数组元素下标
			//数组[i] -- 数组元素
		}
	
	练习：声明一个数组，保存各省份名称，利用循环将所有省份名称前加一个‘★’
	
	4.数组的分类
		索引数组：以0以上的整数为下标的数组
		关联数组：以自定义字符串为下标的数组
	
	5.关联数组
		声明：
			关联数组必须先声明一个空数组，然后挨个往里添加元素
			var arr=[];
			arr['书名']='《西游记》';

		注意：下标为字符串的元素不计入数组长度

		遍历关联数组：for-in
			for(var k in 数组){
				//k -- 数组元素下标
				//数组[k] -- 数组元素
			}
