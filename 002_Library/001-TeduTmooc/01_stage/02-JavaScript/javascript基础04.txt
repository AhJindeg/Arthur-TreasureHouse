一、ES6
   ECMA
   ECMAScript   ES   标准规范
   ES6   第六套标准规范

（1）块级作用域 --- let
   let 声明的变量不存在变量提升，不允许重复声明
   所有大括号之间的语句块都在块级作用域下，里面用let、const声明的是局部的，不允许被外部访问

   面试题：var、let、const   三者的区别
      var 声明的变量存在提升，允许重复声明，只有在函数内声明才是局部变量，不存在块级作用域
      let 声明的变量不存在变量提升，不允许重复声明
      const 声明的常量，声明后必须赋值，不允许重新赋值
      let、const 在函数中和在块级作用域下都是局部的，存在块级作用域
      let、const 在全局作用域声明的也不是全局
      var 在全局作用域声明的是全局

（2）参数增强 --- a=0
   可以给参数设置默认值
   function add(a,b,c=0){
      b = b || 0; // es6 之前设置默认值的方式
   }
   add(8000,2000)
   如果不提供实参，就会使用默认值

（3）箭头函数 --- ()=>{}
   简化了匿名函数的写法，不等价与匿名函数
   sort((a,b)=>b-a); 
   如果箭头函数的函数体之中只有一行代码，并且是return形式，可以进一步简写
   sort((a,b)=>b-a);

（4）模板字符串
   简化之前的字符串拼接
   ` 反引号之间的字符串就是模板字符串 ${JS表达式} `
